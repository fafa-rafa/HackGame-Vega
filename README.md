# HackGame Vega
HackGame Vega is a custom firmware for HackHeld Vega II by Spacehuhn

Games: Snake, Breakout, Shooter
Working on other games too

# INSTALATION
To install HackGame Vega:
1. download the latest firmware from releases page
2. visit esp.huhn.me
3. connect and drag the files
4. click flash
5. ur done :)
